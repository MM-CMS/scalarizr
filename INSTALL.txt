python >= 2.5.1 required 
tested on fedora 8, debian 5, centos 5.4

== CentOS 5.1-5.4/RedHat 5.1-5.4

$ vim /etc/yum.repos.d/IUS.repo
[ius]
name=IUS
baseurl=http://dl.iuscommunity.org/pub/ius/stable/Redhat/5/i386/
enabled=1
gpgcheck=0

$ yum install python26 python26-setuptools gcc
